A professional, responsive personal portfolio website showcasing my skills, projects, and achievements as a MERN Stack Developer.

🚀 Features

· Modern UI/UX – Clean design with gradient themes and smooth animations
· Responsive – Mobile-first approach, works on all devices
· Projects Showcase – 3 live projects with tech stack badges and demo links
· Skills Section – Categorized frontend, backend, database, and tools
· Typing Animation – Dynamic hero section with "MERN Stack Developer | Cybersecurity Enthusiast"
· Dark/Light Mode – Toggle between themes
· Contact Section – Location, email, phone, and social links
· Resume Download – Direct PDF download button
· Interactive Elements – Hover effects, progress bars, smooth scrolling

🛠️ Tech Stack

· Frontend: HTML5, CSS3, JavaScript (ES6+)
· Styling: Custom CSS with variables, Flexbox, Grid
· Icons: Font Awesome
· Fonts: Google Fonts (Poppins/Inter)
· Deployment: Replit / Netlify / Vercel
